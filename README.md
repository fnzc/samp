# SAMP
Simple Async Messaging Protocol

## Development

### Prerequisites
You will need [Gradle][1].

[1]: http://gradle.org

### Testing
Run tests

    gradle test
    
## Publish
set environment variables
    
    export GITHUB_USERNAME=<your github username>
    export GITHUB_TOKEN=<github token>


and run

    gradle publish
